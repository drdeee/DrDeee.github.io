Maubot Plugins

Als totaler Matrix-Fanatiker muss mensch natürlich auch Matrix Bots schreiben. Dafür nutzen wir bei FFF eigentlich immer Maubot, ein pluginbasiertes System in Python für Matrix Bots. dieses Bietet sich einfach an, da damit Hosting & vor allem End to End Encryption abgedeckt sind.

Hier eine Liste von Plugins die ich geschrieben habe:

TodoPlugin: verwaltet für jeden Raum eine Todo-List
TicketSystemPlugin: ein kleines Support/Ticketsystem
HelpPlugin: bietet die Möglichkeit an, Hilfsanweisungen & Dokumentationen zu den einzelnen Plugins zu verwalten
PollPlugin: ein seeehr simples Abstimmungs-Plugin, welches zwar nicht schön aussieht, aber auch über Telegram & WhatsApp Bridges nutzbar ist

Außerdem habe ich ein winziges Scriptchen geschrieben, welches mensch gut in Github Actions nutzen kann, um das Plugin automatisch zu verpacken und als Release zu veröffentlichen. Einen Beispiel-Workflow findest du hier.
